** Bus Ticket Reservation System Using Data Structures
*Objectives
   To automate bus ticket booking and cancellation
   To manage passenger records efficiently
   To implement core data structures in a real-world application
   To improve search efficiency and data handling
*Data Structures Used
   Structure	Purpose
   Binary Search Tree (BST)	Store and search buses
   Linked List
   Stack
*System Features
   Add new buses
   Manage booked passengers
   Handle cancelled seats
   Display available buses
   Book bus tickets
   Cancel booked tickets
   Reallocate cancelled seats
   Display passenger details
*Advantages
    Faster bus search using BST
    Efficient seat reallocation
    No unnecessary waiting list
    Simple and memory-efficient system
*Applications
   Bus transport booking counters
   Online bus reservation systems
   Academic mini-project
   Learning data structures
